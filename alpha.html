Resources

Glowworm

A browser-based bilingual (Spanish/English) toolset leveraging cloud infrastructure for interacting with the public LSST data subset, including an interactive 2D viewer featuring: pan, zoom, contrast / brightness / color / filter controls, light curve data, point-of-interest metadata, associated alerts, and time-series display.

Find out more about Glowworm

Transient Events

A smartphone application that provides alert stream data in a mobile, graphical format. Customizations and object details include: types and brightness, right ascension and declination coordinates, finder chart, and image cutout. The app will initially use events from the Catalina Real Time Survey as precursor data.

Find out more about Transient Events

Educator Hub

A growing set of standards-based educational materials leveraging LSST data for formal and informal education coupled with professional development training to effectively incorporate the curricula into middle school, high school, and undergraduate classrooms as well as informal science centers and planetariums.

Find out more about Educator Hub

Citizen Projects

Harnessing the collective power of citizen scientists from all over the world, key LSST data analysis challenges will be solved by crowdsourcing the work to thousands of volunteers. Projects may potentially include: detection of asteroids, identifying lensing or tidal streams, and analysis of light curve or alert stream data.

Find out more about Citizen Projects

API

A public Application Programming Interface (API) will allow users or apps to subscribe to time-sensitive, customized alerts. This will enable creative and serendipitous uses of the public LSST dataset by independant developers, amateur astronomers, planetarium software companies, and others.

Find out more about API

Communications

The LSST EPO and Communications groups work very closely together to ensure information flows between the LSST team, affiliates, scientific stakeholders, reporting agencies, and the media. Project accomplishments are regularly promoted via social media, conferences, events, and established media outlets.

Find out more about Communications
